# cost_regularized_UOT

This is the official repository of the paper [Structured Matching via Cost-Regularized Unbalanced Optimal Transport](https://arxiv.org/abs/2511.19075).

## Repository Structure

```
.
├── datasets/       
├── src/            # Source code and scripts
└── notebooks/     
```

## Getting Started

```bash
git clone https://github.com/emapardini/cost_regularized_UOT
cd cost_regularized_UOT
conda create -n cruot_env python=3.11.5
conda activate cruot_env
pip install -r requirements.txt
```

## Requirements

- Python 3.11.5
- requirements.txt
